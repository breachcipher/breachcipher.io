Lazy Encryptor menggunakan metode Hybrid Encryption RSA-2048 + AES-256. File korban dienkripsi dengan AES-256, sedangkan kunci AES itu sendiri dilindungi dengan RSA public key. Hanya pemilik private key yang dapat memulihkan akses ke file korban.
